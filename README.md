About perl-file-path-feedstock
================================

Feedstock license: [BSD-3-Clause](LICENSE.txt)

Home: https://metacpan.org/pod/File::Path

Package license: Perl 5

Summary: Create or remove directory trees (`File::Path`).


Build order
===========

Build **`perl-exporter`** before **`perl-file-path`**: `File::Path` declares a runtime dependency on `Exporter`.

Installing perl-file-path
===========================

Installing `perl-file-path` from the main channel:

```
conda install perl-file-path
```

List available versions:

```
conda search perl-file-path
```
